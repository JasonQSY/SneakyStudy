# SneakyStudy

> 偷偷学习时的自动水群机器人

## Dependencies

- `nodejs`
- `npm`

## Usage

```
git clone https://github.com/JasonQSY/SneakyStudy.git && cd SneakyStudy
npm install
node water.js
```

After login, the bot enters an endless loop.
